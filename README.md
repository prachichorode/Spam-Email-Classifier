# 📧 Spam Email Classifier

A Machine Learning based Spam Email Detection System built using Streamlit and Scikit-Learn.

## 🚀 Features

- 🔐 Login System
- 📩 Spam Email Prediction
- 🤖 Multiple ML Models
  - Naive Bayes
  - Logistic Regression
  - SVM
  - Random Forest
- 📊 Analytics Dashboard
- 📈 Confusion Matrix
- 📉 Accuracy Visualization
- ☁️ Spam Word Cloud
- 📂 Dataset Explorer
- 📜 Prediction History
- 📥 Download Prediction Report
- 🚪 Logout Functionality

## 📊 Machine Learning Workflow

1. Load Dataset
2. Clean Text Data
3. Convert Text using TF-IDF Vectorizer
4. Split Dataset into Training and Testing Sets
5. Train Selected Model
6. Predict Spam or Ham
7. Display Analytics and Results
